# UI Requirements

| ID | Status | Requirement |
|----|--------|-------------|
| REQ-801 | ✅ | 使用 HTMX 实现无页面刷新的 SPA 交互 |

## 站点抓取

| ID | Status | Requirement |
|----|--------|-------------|
| REQ-821 | ✅ | 输入框支持域名或完整 URL，回车或点击按钮触发抓取 |
| REQ-822 | ✅ | 进度条：5px 高度，shimmer 流光动画，按阶段递增百分比 |
| REQ-823 | ✅ | 进度阶段对应宽度：waiting=15%、listing=30%、found=45%、navigating=55%、evaluating=80%、done=95% |
| REQ-824 | ✅ | 完成 / 错误 / 取消时进度条变绿 / 红 / 橙，3 秒后恢复 |
| REQ-825 | ✅ | 停止按钮：取消正在进行的抓取任务 |
| REQ-826 | ✅ | 刷新按钮：已存站点可一键更新 Cookie |
